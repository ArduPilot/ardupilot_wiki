==========================
Rover Operational Features
==========================

Rover has a lot of different vehicle types, and advanced features which can be accessed either via
configuration parameters or via mission commands. The list below will
give you some information on some of the more widely used vehicle types and features. For
a full set of features see the :ref:`common-advanced-configuration` page, the :doc:`full parameter list <parameters>`, and the `MAVLink protocol definition <https://mavlink.io/en/>`__.
