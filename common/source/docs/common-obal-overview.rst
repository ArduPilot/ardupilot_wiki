Specifications
==============

-  **Processor** (`Raspberry PI Zero & 42 <https://www.raspberrypi.org/products/>`__)

-  **Sensors**

   -  MPU9250 as main accel, gyro and compass
   -  BMP-180 barometer
   
-  **Interfaces**

   -  UART, SPI, I2C
   -  PWM 8-Channels
   -  16 PWM servo outputs
