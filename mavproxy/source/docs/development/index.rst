===========
Development
===========

This section contains topics to help you get started with
developing MAVProxy and MAVProxy modules. This section assumes
basic knowledge of setting up development environments, Python
and git.

For developing MAVProxy, a slightly more complex setup is
required. The following section will show how this setup is
achieved for Linux and Windows.
